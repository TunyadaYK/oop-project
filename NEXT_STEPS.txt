 ✅ Hotel Reservation System - NEXT STEPS
═══════════════════════════════════════════════════════════════════════════════

📌 YOU NEED TO DO THIS:

1. UPDATE requirements_hotel.txt ✅ (DONE - use version 1.26.0 for mcp)

2. INSTALL DEPENDENCIES:
   In PowerShell (in project folder):
   
   (.venv) PS D:\oop-project> pip install -r requirements_hotel.txt
   
   ✅ This will install:
   - fastapi==0.104.1
   - uvicorn==0.24.0
   - httpx==0.25.2
   - pydantic==2.5.2
   - mcp==1.26.0  (UPDATED from 0.1.8)
   - pyjwt==2.8.1
   - python-dotenv==1.0.0


3. RUN FASTAPI SERVER (Terminal 1):
   
   python hotel_reservation_fastapi.py
   
   Expected output:
   INFO:     Uvicorn running on http://127.0.0.1:8001
   INFO:     Application startup complete


4. CONFIGURE CLAUDE DESKTOP:
   
   Edit: C:\Users\<YourUsername>\AppData\Roaming\Claude\claude_desktop_config.json
   
   Add:
   {
     "mcpServers": {
       "hotel-reservation": {
         "command": "python",
         "args": ["D:\\oop-project\\hotel_reservation_mcp_server.py"],
         "env": {"PYTHONUNBUFFERED": "1"}
       }
     }
   }
   
   (Replace D:\\oop-project\\ with your actual path)


5. RESTART CLAUDE DESKTOP:
   - Close Claude completely
   - Open Claude again
   - Look for "hotel-reservation" in MCP servers


6. TEST IN CLAUDE:
   
   Try saying:
   "Login with username guest1 and password password123"
   
   If it works, you're all set! 🎉
